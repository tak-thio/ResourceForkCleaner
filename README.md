# Resource Fork Cleaner

Macのリソースフォークファイル（`._*`）や `.DS_Store` などの不要ファイルを、指定したフォルダ以下から再帰的に検索し、一括削除するツールです。
WindowsファイルサーバーやNASにMacからアクセスした際に残る「ゴミファイル」の掃除に最適です。

## 特徴
- **高速スキャン**: 5万件以上のファイルがあってもフリーズせずに動作します。
- **安全設計**: 削除前に一覧を表示し、確認ダイアログを経てから削除します。
- **簡単操作**: フォルダをウィンドウにドラッグ・アンド・ドロップするだけでスキャン開始。
- **進捗表示**: プログレスバーで処理状況がひと目で分かります。

## インストールと使い方
1. `dist/ResourceForkCleaner.app` をローカルの任意の場所（アプリケーションフォルダなど）にコピーします。
2. アプリを起動します。（初回はControl+クリックで「開く」を選択）
3. 掃除したいフォルダをウィンドウにドラッグ・アンド・ドロップします。
4. 不要ファイルが検出されたら、「一覧のファイルを削除する」ボタンをクリックします。
